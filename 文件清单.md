## 🚀 使用建议

### 新手入门

1. 先阅读 `README.md` 了解项目
2. 运行 `./setup.sh` 安装依赖
3. 阅读 `使用指南.md` 学习使用
4. 运行 `python demo.py` 查看演示
5. 运行 `python test_connection.py` 测试
6. 运行 `python video_automation.py` 开始使用

### 日常使用

1. 准备视频和提示词文件
2. 运行 `./run.sh` 或 `python video_automation.py`
3. 查看 `automation.log` 了解进度
4. 检查输出文件

### 问题排查

1. 查看 `快速参考.md` 找常见问题
2. 查看 `使用指南.md` 的故障排除章节
3. 查看 `automation.log` 日志文件
4. 查看 `screenshots/` 截图文件

### 配置调整

1. 编辑 `config.py` 修改配置
2. 参考 `使用指南.md` 的配置说明
3. 参考 `快速参考.md` 的常用配置
