# 🚀 快速打包指南

5 分钟将程序打包为可执行文件！

---

## 📋 准备工作

### 确认环境

```bash
# 检查 Python 版本
python --version  # 需要 3.8+

# 检查 pip
pip --version
```

---

## 🎯 三步打包

### 步骤 1: 选择打包方式

根据你的操作系统选择：

#### Windows 用户

```bash
# 双击运行
build.bat

# 或命令行
build.bat
```

#### macOS/Linux 用户

```bash
# 添加执行权限（首次）
chmod +x build.sh

# 运行
./build.sh
```

#### 跨平台（推荐）

```bash
# 使用 Python 脚本
python build.py
```

### 步骤 2: 等待完成

打包过程约 2-3 分钟，会自动：
1. ✅ 检查 PyInstaller
2. ✅ 安装依赖（如需要）
3. ✅ 清理旧文件
4. ✅ 构建可执行文件
5. ✅ 创建说明文件

### 步骤 3: 测试运行

```bash
# 进入输出目录
cd dist/VideoAutomation

# Windows
VideoAutomation.exe

# macOS/Linux
./VideoAutomation
```

---

## 📦 输出结果

### 文件位置

```
dist/
└── VideoAutomation/
    ├── VideoAutomation.exe  (Windows)
    ├── VideoAutomation      (macOS/Linux)
    └── _internal/           (依赖文件)
```

### 文件大小

- Windows: ~80MB
- macOS: ~75MB
- Linux: ~70MB

---

## 🎉 完成！

### 分发

压缩 `dist/VideoAutomation` 文件夹：

```bash
# Windows
cd dist
tar -czf VideoAutomation-v1.3.8-windows.zip VideoAutomation/

# macOS/Linux
cd dist
tar -czf VideoAutomation-v1.3.8.tar.gz VideoAutomation/
```

### 使用

用户只需：
1. 解压文件
2. 双击运行（Windows）或 `./VideoAutomation`（Mac/Linux）
3. 无需安装 Python！


### 打包失败

查看详细文档：
- [打包说明.md](打包说明.md)

1. 查看日志输出
2. 检查 Python 版本
3. 确认所有依赖已安装

## 💡 提示

### 首次打包

- 需要下载依赖，可能较慢
- 确保网络连接稳定

### 再次打包

- 会自动清理旧文件
- 速度更快

### 自定义配置

编辑 `build.spec` 文件：
- 添加图标
- 修改名称
- 调整配置
