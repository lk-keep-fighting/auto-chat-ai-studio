### 步骤25数据保存测试工具 🧪

快速测试步骤25的数据提取和保存功能。

## 快速使用

### 运行测试

```bash
# 方法1：使用启动脚本
./test_step25.sh

# 方法2：直接运行
python test_step25.py
```

### 测试流程

1. 自动打开 AI Studio
2. 等待用户确认（按 Enter）
3. 自动发送测试提示词
4. 等待AI生成10行测试数据
5. 自动提取表格数据
6. 自动保存为Excel
7. 验证并显示结果

### 查看结果

```bash
# 查看Excel文件
open test_output/test_step25/step_25_output.xlsx

# 查看日志
cat test_step25.log
```

## 故障排除

### 数据量为0

```
📊 数据量: 0
❌ 未提取到任何数据
```

**解决**：AI还没生成完，手动等待后再按Enter

### 数据类型是字符串

```
📊 数据类型: <class 'str'>
```

**解决**：DOM提取失败，检查HTML结构

### 文件未创建

```
❌ 文件未创建: step_25_output.xlsx
```

**解决**：查看日志中的错误信息

### 文件大小为0

```
📊 文件大小: 0 字节
```

**解决**：DataFrame为空，检查数据提取逻辑
