## 🚀 部署说明

### 系统要求

- **操作系统**: macOS / Linux / Windows
- **Python 版本**: 3.8 或更高
- **网络要求**: 稳定的互联网连接
- **浏览器**: Chromium（自动安装）
- **磁盘空间**: 至少 500 MB

### 安装步骤

1. **克隆或下载项目**
   ```bash
   # 如果是 Git 仓库
   git clone <repository-url>
   cd <project-directory>
   ```

2. **运行安装脚本**
   ```bash
   chmod +x setup.sh
   ./setup.sh
   ```

3. **准备数据文件**
   - 确保 `assets/Process_Folder/videos/VideoList.csv` 存在
   - 确保 `assets/Process_Folder/prompts.xlsx` 存在
   - 确保视频文件在 `assets/Process_Folder/videos/` 目录

4. **运行程序**
   ```bash
   python video_automation.py
   ```

### 首次运行

1. 程序会自动打开浏览器
2. 需要手动登录 Google 账号
3. 完成授权后程序自动继续
4. 登录信息会保存，后续无需重复登录

## 🔧 配置说明

### 主要配置项

编辑 `config.py` 文件：

```python
# 浏览器模式
HEADLESS = False  # True=无头模式，False=显示浏览器

# 等待时间（秒）
WAIT_AFTER_UPLOAD = 15    # 上传视频后等待
WAIT_FOR_RESPONSE = 10    # AI 响应等待

# 日志和截图
LOG_LEVEL = "INFO"        # 日志级别
SAVE_SCREENSHOTS = True   # 是否保存截图

# 重试配置
MAX_RETRIES = 3          # 最大重试次数
RETRY_DELAY = 5          # 重试延迟（秒）
```

### 选择器配置

如果 Google AI Studio 页面结构变化，需要更新选择器：

```python
SELECTORS = {
    'file_upload': 'input[type="file"]',
    'input_box': 'textarea[placeholder*="Enter"]',
    'send_button': 'button[aria-label*="Send"]',
    # ... 更多选择器
}
```

## 🐛 故障排除

### 常见问题

1. **找不到上传按钮**
   - 解决方案：更新 `config.py` 中的选择器

2. **视频上传超时**
   - 解决方案：增加 `WAIT_AFTER_UPLOAD` 的值

3. **AI 响应慢**
   - 解决方案：增加 `WAIT_FOR_RESPONSE` 的值

4. **首次需要登录**
   - 这是正常的，手动登录后会自动保存

### 调试方法

1. 查看日志文件：`cat automation.log`
2. 查看截图文件：`screenshots/` 目录
3. 运行测试脚本：`python test_connection.py`
4. 启用调试模式：编辑 `config.py` 设置 `DEBUG_MODE = True`
