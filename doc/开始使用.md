# 🚀 开始使用 - 5 分钟快速上手

## 第一步：安装依赖（2 分钟）

打开终端，进入项目目录，运行：

```bash
chmod +x setup.sh
./setup.sh
```

或者手动安装：

```bash
pip install -r requirements.txt
playwright install chromium
```

## 第二步：准备文件（1 分钟）

确保以下文件存在：

### 1. 视频列表文件

位置：`assets/Process_Folder/videos/VideoList.csv`

格式：
```csv
Filename,Duration
video1.mp4,0:12:57
video2.mp4,0:15:30
```

### 2. 提示词文件

位置：`assets/Process_Folder/prompts.xlsx`

包含列：
- 文件名称
- 视频时长
- 步骤1 到 步骤25（共 25 个提示词）

### 3. 视频文件

位置：`assets/Process_Folder/videos/`

确保视频文件名与 `VideoList.csv` 中的一致。

## 第三步：运行脚本（1 分钟）

```bash
python video_automation.py
```

或使用快捷脚本：

```bash
chmod +x run.sh
./run.sh
```

## 第四步：运行并确认

1. 脚本会自动打开系统 Chrome 浏览器
2. 自动访问 Google AI Studio
3. **等待你确认已登录并进入对话界面**
4. 按 Enter 键后脚本继续处理

**确认步骤**：
```
============================================================
👤 请确认以下操作
============================================================
1. 确保已登录 Google 账号
2. 进入 AI Studio 对话界面
3. 准备好开始处理视频
============================================================

✅ 完成上述操作后，按 Enter 键继续...
```

**注意**：
- ✅ 默认使用系统 Chrome 和默认用户配置
- ✅ 如果 Chrome 已登录 Google 账号，直接可用
- ✅ 需要手动进入对话界面
- ✅ 确认后脚本自动处理所有视频

**如果不想每次确认**：
```python
# 修改 config.py
WAIT_USER_CONFIRMATION = False
```

## 第五步：等待完成

脚本会自动：
1. ✅ 读取视频列表
2. ✅ 更新提示词文件
3. ✅ 上传视频
4. ✅ 执行 25 步对话
5. ✅ 保存输出数据
6. ✅ 合并所有文件
7. ✅ 运行最终处理

## 🎯 常见问题

### Q: 找不到视频文件

**A:** 检查路径是否正确：
```bash
ls assets/Process_Folder/videos/
```

### Q: 上传很慢

**A:** 这是正常的，视频上传需要时间。可以增加等待时间：

编辑 `config.py`：
```python
WAIT_AFTER_UPLOAD = 30  # 增加到 30 秒
```

### Q: 需要看到浏览器操作

**A:** 默认会显示浏览器窗口。如果没有，编辑 `config.py`：
```python
HEADLESS = False
```

### Q: 出现错误怎么办

**A:** 
1. 查看 `automation.log` 日志
2. 查看 `screenshots/` 截图
3. 运行测试脚本：
   ```bash
   python test_connection.py
   ```
