## 💡 提示

### 快速查找

使用 Ctrl+F (或 Cmd+F) 在文档中搜索关键词：

- 搜索 "安装" → 找安装方法
- 搜索 "配置" → 找配置说明
- 搜索 "错误" → 找错误处理
- 搜索 "示例" → 找代码示例

### 文档链接

所有文档都包含相互链接，可以方便地跳转：

- 点击文档中的蓝色链接
- 跳转到相关文档
- 查看更多详细信息

### 实用工具

```bash
# 查看所有文档
ls *.md

# 搜索文档内容
grep -r "关键词" *.md

# 查看文档大小
ls -lh *.md
```
